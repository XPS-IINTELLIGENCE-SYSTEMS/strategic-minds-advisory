# AI in Action Labs Wiki

## Vision

AI in Action Labs is a public learning platform where AI demonstrates useful work in real time.

The platform is designed to let people watch AI research, simulate, build, validate, explain, and package strategies for learning, business creation, paper trading, content production, and digital systems.

The point is not to pretend AI is magic. The point is to show the actual process:

```text
idea -> research -> source receipts -> simulation -> build -> validation -> content -> public lesson
```

## Core Mission

Create a controlled autonomous AI environment that can:

- run 24/7 scheduled simulations
- build digital business experiments
- run paper-trading education labs
- generate AI avatar scripts and content
- validate sources
- log every important action
- show human-readable dashboards
- teach viewers what is happening and why
- ask humans for approval when needed

## First Flagship Lab

### AI Paper Trading Lab

A simulated $50,000 paper account designed to teach market literacy, risk management, source verification, no-fake-fill discipline, portfolio simulation, trade journaling, educational reporting, and no-guarantee wealth-building lessons.

Rules:

- paper trading only
- no financial advice
- no fake live prices
- no fake fills
- no guaranteed outcomes
- verified source links required

```text
GitHub Project = command board
Vercel = frontend and serverless deployment
Supabase = backend database, auth, storage, logs
Groq = AI generation route
Google Drive = operating documents
Gmail = reports and human summaries
GPT Scheduler = recurring educational/report layer
```

## Operating Philosophy

AI should be visible, truthful, source-backed, controlled, useful, educational, iterative, and safe.

AI should not be secretive, uncontrolled, deceptive, fake, overclaiming, financially reckless, or dependent on one fragile chat session.
